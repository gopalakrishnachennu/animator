# Animation

Minimal, clean, and playful animation sandbox with a focus on smooth visuals and simple structure.

![Preview](assets/image.png)

## Mini List
- Quick setup with `npm install` and `npm run dev`
- Vanilla JS + CSS animations for easy tinkering
- Lightweight project layout
- Ready to customize scenarios and assets

## Quick Start
```bash
npm install
npm run dev
```

## Project Structure
- `index.html` — entry page
- `style.css` — styling and animation rules
- `main.js` — core animation logic
- `assets/` — images and static assets
- `scenarios/` — animation scenarios
